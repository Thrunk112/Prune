# Prune
V1 Beta test

Prune aims to keep you below buff cap by clicking off the lowest prio buff when you reach 30 buffs. 

Costumes are commented out for preformance testing. Some icons may click off unintended buffs, please report any of these.

Buffs are clicked off in order from the RemoveNearCap list. Add or move things as you see fit. Buffs can also be added to the Always remove list. Currently it is set up for my warrior with int and BoP always being removed.
